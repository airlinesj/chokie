# Assets Folder - Add Your Images Here

## Required Images

Place the following images in this folder:

### 1. chokie-main.jpg
- **Purpose**: Main image on landing page
- **Recommended Size**: 250x250px (square)
- **Format**: JPG or PNG
- **Description**: The main birthday photo that appears on the landing page

### 2. chokie-1.jpg through chokie-5.jpg
- **Purpose**: Gallery images on appreciation page
- **Recommended Size**: 500x500px (square aspect ratio)
- **Format**: JPG or PNG
- **Description**: Photos that will appear in the sliding gallery with captions

## How to Add Images

1. Navigate to the `src/assets/` folder in your project
2. Add your images with the exact names listed above
3. Make sure images are in JPG or PNG format
4. Recommended: Use square aspect ratios for best appearance

## Image Captions

The gallery images have default captions that appear on hover:
- chokie-1.jpg: "Your beautiful smile lights up the room"
- chokie-2.jpg: "Every moment with you is precious"
- chokie-3.jpg: "The strength you show inspires us"
- chokie-4.jpg: "Together through thick and thin"
- chokie-5.jpg: "Friends forever, always by your side"

You can edit these captions in `src/app/pages/appreciation/appreciation.component.ts` in the `photos` array.

## File Sizes

Keep file sizes optimized for web:
- Images under 500KB each
- Use jpg for photos (smaller file size, good quality)
- Use png for transparent images

## Tips for Best Results

✨ **Best for Gallery Photos**:
- Use high-quality photos with good lighting
- Square aspect ratio works best
- Include a variety of moments (candid, posed, group photos)

📸 **For Landing Image**:
- Should be a clear, face-forward photo
- Good lighting is important
- Will be displayed at 250x250px
